# Sonoria — Docs

Documentation et notes d'architecture du monorepo.

## Structure

- `apps/front` — Frontend Next.js (App Router)
- `apps/gateway` — API Gateway NestJS
- `apps/users` — Microservice Users (NestJS)
- `packages/eslint` — Configs ESLint partagées
- `packages/tsconfig` — TSConfigs partagées
- `packages/schemas` — Schémas / DTO partagés
- `packages/ui` — Composants UI partagés
- `packages/utils` — Utilitaires partagés
